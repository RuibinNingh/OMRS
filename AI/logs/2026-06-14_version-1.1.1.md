# 2026-06-14 版本号更新到 v1.1.1

## 更新概要

v1.1.1 是一次历史记录体验和可靠性修复版本。重点修复 Ledger 历史修正里“撤销后恢复不真正生效”的投影问题，历史页默认进入只读浏览，修正操作需要显式开启修正模式；主时间线也改为题目优先展示，让复盘时能直接看到是哪一道题发生了反馈或修正。

## 主要变化

- 历史恢复更可靠：`review.restore` 和 `session.restore` 现在会从原始反馈提交重新回放，正确恢复 `history_log.csv`、mastery 和题目状态。
- 历史修正更安全：历史修正 API 在追加 commit 前校验目标反馈、Session、目标 seq 和替换分数，避免无效修正污染 Ledger。
- 历史页更清楚：修正记录独立展示，主时间线按题目 UID 优先显示；对错统计、有效题数和 Session ID 放在副标题。
- 历史页更防误触：默认关闭修正模式，只有开启后才显示修改、撤销、还原等危险操作。
- 设置页图片优化体验继续完善：快扫/深扫分阶段展示进度，降低等待时的不确定感。

- `AI/README.md`、`AI/api.md`：同步当前版本说明。
- `AI/logs/log.md`：补充版本日志索引。
