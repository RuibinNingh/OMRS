📥 JSON 导入与录入队列

不配置 API 也能用 AI 录题：复制提示词 → 发给任意 AI（附上题目/答案截图）→ 把它回复的 JSON 粘贴到下方导入。草稿进入队列后可分多次逐条「填入表单」配图创建，或纯文字「直接创建」；队列保存在本机浏览器，关页不丢。

📖 使用说明

1. 在「题目图片」区 Ctrl/⌘+V 粘贴题目截图（也可拖拽 / 点击选择）

2. 点「🤖 提取并填充信息」→ AI 自动填好科目 / 分类 / 难度（不抄题）

3. 答案可在「答案图片」区粘图后点「🤖 提取答案」转成文本，或手动输入 / 直接留图

4. 核对各字段，点「创建题目」即写入；题目与答案图片都会嵌入对应区域

提示：系统自动编号，文件为 错题/科目/分类/分类N.md，图片存入 错题/附件/。AI 需先在「设置」中配置。

无 API 方案：用上方卡片「复制 AI 提示词」→ 任意 AI 配截图生成 JSON → 导入队列分批创建。中途要走？点「暂存到队列」存草稿，回来继续。

⚙️ 服务设置
开启后，其他设备可通过本机的局域网 IP（如 192.168.x.x:端口）访问；本机始终用 127.0.0.1。不要在浏览器输入 0.0.0.0（它只是「监听所有网卡」的占位地址，无法访问）。修改后需重启生效，启动时控制台会打印可访问的网址。
🤖 AI 自动识别

配置 OpenAI 兼容接口（/v1/chat/completions 协议）后，即可在「录入题目」页粘贴图片用 AI 自动识别填充。需使用支持图片输入的多模态模型。

填到 /v1 即可（会自动补 /chat/completions）。例：OpenAI https://api.openai.com/v1；阿里云百炼 https://dashscope.aliyuncs.com/compatible-mode/v1。
仅保存在本机 错题/.omrs/config.json，调用时由本地后端转发，不经第三方。
开启（默认）：AI 识别题目时，相关知识点只能取自「已有分类 ∪ 已有知识点」，模型造的新词会被剔除——标签更统一、便于按知识点筛选复盘。关闭：当没有贴切的已有项时，允许 AI 新建知识点（最多 4 个，仍优先复用已有）。仅影响知识点；科目 / 分类一直允许在无合适项时新建。
